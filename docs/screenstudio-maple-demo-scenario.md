Click the final Build button.

## Clean Take Strategy

Best live recording:

1. Record scenes 1-4 in the ready-to-build workspace.
2. Let Build wiki run.
3. If the generated result is clean, continue with scenes 5-9.
4. If the generated result is slow or messy, cut after the progress shot and reopen the finished fallback workspace for scenes 5-9.

## ScreenStudio Notes

- Use a medium zoom level that keeps the left tree, center page, and right panel visible.
- Use 2-3 zoom-ins only: source list, Build wiki modal, generated study path.
- Keep captions short. Do not explain every UI control.
- Hide desktop clutter before recording.
- Use the actual Maple app, not localhost or browser preview.
